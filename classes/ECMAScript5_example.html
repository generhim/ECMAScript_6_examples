ECMAScript 5 Example: Using a "class-like" structure to mimic traditional classes.

  1. ECMAScript 1 - 5 does not have classes.
  2. ECMAScript 1 - 5: Instead of using classes, people relied on creating a function object -- the "constructor" function -- and then assigning properties to this object and adding functionality to the that object's prototype for later use.
  3. Terminology like "class" where not used. Rather the object was created and then its properties and its prototype were extended to provide the desired results.
  4. CAVEAT: While the term, "class" was not used, we would still create a new instance of this "class-like" function by using the "var NameOfInstance = new NameOfClass()" format.
